# Proyecto JavaScript - Aplicación de Consola

## Descripción
Aplicación desarrollada en JavaScript que permite realizar operaciones básicas,
usar condicionales, ciclos, funciones, arreglos y objetos.

## Cómo ejecutar
1. Abrir el archivo index.html
2. Presionar F12
3. Ir a la pestaña Consola

## Funcionalidades
- Operaciones matemáticas
- Uso de arreglos y filtros
- Creación de objetos
- Ciclos for
- Código accesible para lectores de pantalla
